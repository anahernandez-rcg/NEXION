Connect to Microsoft Fabric

Sign in with your Microsoft account to load live capacity and workspace data from your tenant.

How it works
1. Enter your Azure AD App (Client) ID — a public client registration with no secret required.
2. Click Sign in with Microsoft — you'll be redirected to Microsoft login.
3. After authenticating, NEXION loads your capacities and workspaces live.
Register a public client app in Azure AD with Power BI API delegated permissions: Capacity.Read.All, Workspace.Read.All, Tenant.Read.All
⚠ Before connecting — your Azure Function needs CORS enabled:
1. Azure Portal → your Function App → CORS
2. Add your page origin (or * for testing)
3. Check Enable Access-Control-Allow-Credentials → Save
Azure Function URL — no trailing slash. Function must be named token and accept POST /api/token
